Функциональные клавиши
     Для удобства взаимодействия с программой были добавлены следующие функциональные клавиши:
• F1 – Открыть пользовательскую справку;
• Ctrl + 1 – Открыть панель “Характеристика”;
• Ctrl + 2 – Открыть панель “Личные данные”;
• Ctrl + 3 – Открыть панель “Качества”;
• Ctrl + 4 – Открыть панель “Дополнительно”;
• Ctrl + O – Открыть файл;
• Ctrl + S – Сохранить в файл;
• Ctrl + P – Печать содержимого текстового поля;
• Ctrl + F – Форматирование;
• Ctrl + C – Отмена Форматирования;
• Ctrl + T – Настройка шрифта
